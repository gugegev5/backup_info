## 角度单位

90deg = 100grad = 0.25turn  ≈ 1.570796326794897rad

即：1turn = 360deg = 400grad = 2pi rad

1. deg：度（全称：degress）一个圆有360度

2. grad：梯度（gradians）：一个圆共有400个梯度

3. turn：转、圈（turns）：一个圆共1个圈

4. rad：弧度（radians）：一个圆共2π弧度

}
